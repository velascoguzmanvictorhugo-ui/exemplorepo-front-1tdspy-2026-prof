# 🚀 Atividade Prática: Git Flow e Estilização CSS3

Bem-vindo ao desafio de Front-end! Esta atividade foi desenhada para que você pratique o fluxo real de trabalho em equipe utilizando **Git**, **GitHub** e **CSS3 moderno**.

## 🎯 O Desafio
Sua missão é estilizar o **Cabeçalho (Header)** deste projeto. O HTML já está estruturado, mas o design visual depende inteiramente do seu código CSS.

---

## 🛠️ Diretrizes de Design
Para garantir a consistência do projeto, siga estes requisitos:

* **Layout:** Utilize `flexbox` ou `grid` para alinhar os itens do menu.
* **Cores Sugeridas:** * Fundo do Header: `#2c3e50`
    * Texto/Links: `#ecf0f1`
    * Destaque (Hover): `#3498db`
* **Tipografia:** Use fontes sem serifa (Ex: `Arial`, `Helvetica`, `sans-serif`).
* **Interatividade:** Adicione uma transição suave (`transition`) ao passar o mouse sobre os links.
* **Responsividade:** O menu deve se ajustar ou ser legível em dispositivos móveis.

## ⚙️ Instruções para Entrega (Passo a Passo)

Siga rigorosamente estes passos para que sua contribuição seja aceita:

### 1. Preparação
1. Clique no botão **Fork** (no canto superior direito desta página) para criar uma cópia do projeto na sua conta.
2. Clone o **seu fork** para o seu computador:
   ```bash
   git clone [https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git](https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git)

### 2. Desenvolvimento

1. **Crie uma nova branch** para organizar seu trabalho:
   ```
   git checkout -b feat/estilo-header-SEUNOME
```
1. Abra a pasta no VS Code e edite o arquivo css/style.css.

2. Salve o progresso com um commit descritivo:
```   
    git commit -m "feat: estilização do cabeçalho finalizada"
```
3. Envie a branch para o seu GitHub:
```
    git push origin feat/estilo-header-SEUNOME
```

4. Finalização
